# 🚀 AI Interview Preparation Platform

An intelligent full-stack web application designed to help users prepare for technical interviews using AI-driven insights, resume analysis, and adaptive question generation.

---

## 📌 Overview

This platform simulates real interview scenarios by analyzing user resumes and generating tailored interview questions. It provides instant feedback, performance scoring, and dynamic follow-up questions to improve candidate readiness.

---

## ✨ Key Features

* 📄 **Resume Parsing & ATS Analysis**
  Upload resumes (PDF) and get insights similar to real ATS systems.

* 🤖 **AI-Based Question Generation**
  Generates interview questions based on:

  * Resume content
  * Selected topics
  * Difficulty level

* 🔁 **Smart Follow-up Questions**
  Dynamically adapts based on user responses.

* ⏭️ **Skip & Next Question Functionality**
  Allows flexible interview flow.

* 📊 **Performance Scorecard**
  Evaluates answers and provides structured feedback.

* 🎯 **Topic-Based Practice Mode**
  Focus on specific subjects like DSA, DBMS, OS, etc.

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/wewakejoshi/ai-interview-platform.git
cd ai-interview-platform
```

### 2. Setup Backend

```
cd backend
npm install
npm start
```

### 3. Setup Frontend

```
cd frontend
npm install
npm start
```
